# CTS BPO – Brand Kit

## Logo Usage

The CTS BPO logo should be placed in `branding/logo.png`.

- Use the full logo on all external-facing documents and the web application header.
- Minimum logo size: 120px wide (digital), 30mm wide (print).
- Maintain clear space equal to the logo height on all four sides.
- Do not distort, rotate, or apply effects to the logo.

---

## Color Palette

| Color Name | Hex | Usage |
|-----------|-----|-------|
| CTS Blue | `#1e40af` | Primary brand color, buttons, headings |
| CTS Green | `#22c55e` | Success states, active badges |

| CTS Red | `#ef4444` | Error states, failed badges |
